Purpose of Project

The purpose of this project is to create a web application that is intended to be functional and purposeful. This application will incorporate various features that have been covered in class such as text, links, buttons, photos, and color. We hope that in creation of this web application we learn new features and topics that we can use in the future.

The web application will be a home page for various National Parks around the country. Theoretical Background

For this project, we will use the knowledge gained from class in order to create the web application. Our goal with the topic of National Parks is to show how Computer Science can bridge the gap between two very different topics: technology and the outdoors.
